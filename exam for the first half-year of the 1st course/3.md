## Лекция 3. Области видимости и операторы ветвления

**Области видимости переменных:** Область видимости (scope) – часть программы, где имя переменной или функции «видно» (может использоваться). В C++ есть:

* **Локальная область (автоматические переменные):** переменные, объявленные внутри блока `{...}`, видимы только внутри этого блока. Их память обычно на  *стеке* . После выхода из блока переменная уничтожается.
* **Глобальная область:** переменные, объявленные вне всех функций, видны по всему модулю (если не скрыты `static`) и существуют до конца программы.
* **Статическая область:** статические локальные переменные (`static` внутри функции) и глобальные (без `extern`) имеют статическую продолжительность – сохраняют значение между вызовами и видимы с места объявления далее по файлу.

**Скрытие имён (shadowing):** Если во вложенном блоке объявить переменную с тем же именем, что и снаружи, внешняя переменная «скрывается» внутри этого блока. В таком случае имя относится к самой близкой области. Например:

```C++
int x = 10;      // глобальная x
void f() {
    int x = 5;   // локальная x скрывает глобальную внутри f()
    std::cout << x;  // выведет 5
}
// после выхода x снова = 10

```

**Операторы условного ветвления:**

* **if / else:** Базовый оператор ветвления. Синтаксис:

  ```C++
  if (выражение) {
      // блок, если выражение истинно
  } else if (другое_выражение) {
      // блок, если первое ложно, второе истинно
  } else {
      // блок, если все условия ложны
  }

  ```

  Выражение в `if` преобразуется к булеву типу: ненулевое считается `true`, ноль – `false`. Пример:

  ```C++
  int n = ...;
  if (n > 0) {
      std::cout << "Positive";
  } else if (n == 0) {
      std::cout << "Zero";
  } else {
      std::cout << "Negative";
  }
  ```

  Оператор проверяет условие и выполняет только один блок.
* **switch / case:** Альтернатива множественному `if` для целочисленных выражений. Пример:

  ```C++
  int code = ...;
  switch (code) {
      case 1:
          std::cout << "One";
          break;
      case 2:
      case 3:
          std::cout << "Two or Three";
          break;
      default:
          std::cout << "Other";
  }
  ```

  Выражение (`code`) должно быть целочисленным (`int`, `char`, `enum` и т.д.). Метка `default:` опциональна – срабатывает, если ни один `case` не подошёл. Без `break` управление «проваливается» к следующему кейсу, что часто используют для объединения случаев.
* **Тернарный оператор `?:`:** Упрощённый `if`. Синтаксис:

  ```C++
  result = (условие) ? значение1 : значение2;
  Это эквивалентно конструкции `if…else`. Работает только с тремя операндами.
  ```

  Если `условие` истинно, берётся `значение1`, иначе – `значение2`. Например:

  ```C++
  int a = 5, b = 10;
  int max = (a > b) ? a : b; // max = 10
  ```

  Это эквивалентно конструкции `if…else`. Работает только с тремя операндами

**Пример комбинации:**

```C++
int score = ...;
if (score >= 90) {
    std::cout << "Grade A";
} else if (score >= 75) {
    std::cout << "Grade B";
} else {
    switch (score / 10) {
      case 6: std::cout << "Grade C"; break;
      default: std::cout << "Grade D or F";
    }
}
```
